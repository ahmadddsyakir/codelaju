FAQ

Frequently Asked Questions

Web development is the building and maintenance of webpages or websites. It is the actual work that happens behind the scenes to make a website look great, work fast and perform well with a seamless user experience. Web development can be broken down further into front-end development and back-end development.

You should have an understanding of HTML, CSS, and JavaScript. It is also highly recommended to learn about CSS and CSS frameworks. Developing these fundamental web development skills will give you the foundation and logic for building interactive and responsive websites.

Due to the ride of e-commerce, web and mobile applications, the demand for web developers in Singapore and around the world is only expected to grow.

The average front end salary in Singapore is S$69,000 per year or S$35.38 per hour. Entry-level positions start at S$49,200 per year, while most experienced workers make up to S$96,000 per year. Reference

You will learn how to build responsive webpages using HTML5 and CSS3. You will also understand how the web works, and the importance of using good design principles. You will have the opportunity to work with a selected CSS framework. In this course, you will also be learning to use GitHub as a start to build your portfolio for prospective employers.

Upon completion of the HTML and CSS in the introduction to front-end development course, you will have completed setting up a portfolio on GitHub and will also be prepared for technical job interviews. We will guide you to the process of updating your linkedin profile and to write in to local related industries for possible internship or entry level jobs.

In the event, that you have already completed the HTML and CSS course and have not been shortlisted for any interviews, continue to upskill yourself to learn Javascript in order to increase the chances of your getting noticed by prospective employers. You can enroll in our earliest Javascript course intake.

This course is designed to cater for absolute beginners with no relevant technical experience on web development. The participant should be equipped with basic computer knowledge.
